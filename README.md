# 오늘의 직장인 하찮운세 🥠

포춘쿠키를 깨면 오늘 하루치 "하찮은" 직장인 운세를 뽑아주는 정적 웹 페이지입니다.
빌드 도구나 서버 없이 HTML/CSS/JS 파일만으로 동작합니다.

## 기능

- **포춘쿠키 깨기**: 쿠키를 클릭(또는 Enter / Space)하면 스프라이트 애니메이션과 함께 쿠키가 갈라지고 이모지 컨페티가 터집니다.
- **하루 한 번 고정 운세**: 그날 처음 뽑은 운세를 `localStorage`에 저장해, 같은 날 다시 방문하면 같은 운세를 보여주고 "내일 다시 시도해 주세요" 안내를 띄웁니다.
- **공유**: `navigator.share`(모바일)로 쿠키 + 운세 이미지를 공유하고, 지원하지 않는 환경에서는 PNG 다운로드 + 텍스트 클립보드 복사로 폴백합니다. 공유 이미지는 `<canvas>`로 즉석에서 그립니다.
- **접근성**: 키보드 조작, ARIA 라벨, `prefers-reduced-motion` 시 컨페티 비활성화.

## 파일 구성

| 파일 | 설명 |
| --- | --- |
| `index.html` | 마크업 + 운세 목록 + 전체 로직(애니메이션, localStorage, 공유) |
| `style.css` | 스타일. Pretendard / Inter / Playfair Display 웹폰트 사용 |
| `cookie_sprite_.png` | 2×3 격자의 쿠키 깨지는 단계별 스프라이트 시트 |

## 실행 방법

`index.html`을 브라우저로 열기만 하면 됩니다. 웹폰트와 CDN을 쓰므로 가능하면 간단한 로컬 서버로 띄우는 걸 권장합니다.

```bash
python3 -m http.server
# http://localhost:8000 접속
```
